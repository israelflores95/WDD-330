Javascript testing in console

    Window Object

  1. The window object can access all things in the browser
    • alerts
    • window sizing
    • adjust window size
    • browser history
    • browser type
  2. The window object is a global veriable...this includes
    • alert (windows.alert and alert are the same)
    • The windows object is where the global scope varibles go
    • Global scope variables are properties of the window object
  3. Windows.history.....controls the history of the browser you can manipulate the history with this.
  4. The use of cookies can be helpful for storing small bits of data
    • The maximum size a cookie can be is 4kb
    • only 20 cookies are allowed per domain
  5. alerts can be annoying...Dont use them if possible.
